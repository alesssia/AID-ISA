void Bicluster::initializeSignature ( unsigned int  num_genes)

Return the initial gene seed for ISA.

First, it sets the gene signature to have an entry for each gene in the matrix. Note that it is not necessary to set the dimension of condition signature: it will be evaluated in each SA run. Second, it randomly select a number of gene indices and set them to an uniform value.

Both the number of genes and the uniform value are global parameter.

Return the results of the AID-ISA algorithm [Visconti et al., Intelligent Data Analysis, 2013].

It evaluates the AID-SA algorithm until the convegence criteria is reached (i.e., the element in both the gene and the condition does not change) or until the initial seed is proved to be divergent (i.e. the number of iteration exceded a global parameter). In the latter case a void bicluser is returned.

Return the results of a run of the AID-SA algorithm [Visconti et al., Intelligent Data Analysis, 2013].

It identifies a transcriptional module given a normalized gene expression matrix and its traspose. AID-SA starts from a gene cluster and it evaluates the correspondant condition cluster. Then, the obtained condition cluster is thresholded and it is exploited to evaluate the new gene cluster. The gene cluster is also thresholded.

See Also
Cluster.calculate, for a detailed description of these steps.
